# SmartBrolly

This project contains a platform focused on making cities more livable through shared umbrella services.

Built using **Next.js**, **Tailwind CSS**, and **shadcn/ui components**, this page highlights the company’s mission, values, and story.

---

##  Features

-  Modern responsive UI with Tailwind CSS
-  Component-based architecture using Next.js
-  Clean card layout for core values
-  Fully responsive (mobile, tablet, desktop)
-  Optimized images using Next.js `Image` component
-  Clear storytelling structure (Mission → Values → Story)

---

## Tech Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui (`Card`)
- **Icons:** lucide-react
- **Image Optimization:** next/image
